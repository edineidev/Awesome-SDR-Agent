# Awesome SDR Agent

## About
Awesome SDR Agent is a chatbot that can help you to manage your WhatsApp account.

![Diagram](doc/diagram.png)

## Stack
- Python
- LLM (OpenAI)
- WhatsApp (Evolution API)
- Docker

## run local with docker
1. `echo "OPENAI_API_KEY=YOU-OPENAI_API_KEY" >> .env`
2. `docker-compose up`
